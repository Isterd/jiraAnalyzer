ClientConfig analyzer

Данное приложение позволяет осуществить аналитику по проектам из открытых источников ClientConfig.

Добавить проекты для аналитики вы можете на странице "Проекты".

Посмотреть статистику по скачанным проектам и выполнить аналитические задачи для выбранных проектов вы можете через страницу "Добавленные проекты". Результат выполнения аналитических задач будет представлен в виде графика.

Это будет гистограмма, отражающая время, которое задачи провели в открытом состоянии.

Провести сравнение по проектам можно перейдя на страницу "Сравнение". Обратите внимание на то, что в результате сравнения проектов будут построены только те сводные диаграммы, аналитика, для которых вызывалась у всех проектов, которые вы хотите сравнить.
